Introduction

What is Reddit?

Reddit is known as the front page of the internet. It is a social news aggregation, web content rating and discussion website. There are many different subreddits or discussion groups that appeal to a wide variety of users. Whether you love cute animals (/r/aww) or just want to see the news (/r/news) or to even things like cats standing up (/r/CatsStandingUp), there exists a subreddit for you. You can use it to read the news during breakfast, or get help or suggestions with your hobbies or just plain procrastinating. Reddit will provide you with endless hours of entertainment.

Visit reddit

What is an API?

An API, otherwise known as an application program interface, is a set of routines, protocols and tools for building software applications. An API consists of building blocks to specify how software components should interact with eachother. A programmer can then use these blocks to create a wide variety of different programs. There exists a variety of different types of APIs for different applications. Today, we will learn how to use the Reddit API

Method #1 - Getting to know ajax, json and jquery

For this method, an API key is not required. This is a simple method for us to learn the basics.

Ajax is a client side script that communicates with a server or database that doesn't need a complete page refresh. In simple terms, it is a method for exchanging data with a server and updating parts of the webpage without refreshing the page.

JSON on the other hand is a syntax for storing and exchanging data between a client and server. It is used as a means to structure data in a readable format.


jQuery is a javascript library that makes it easier to do things like HTML document traversal and manipulation, event handling, animation, and Ajax simpler

Method #1 - Breakdown: Part 2
document.getElementById("subsend").addEventListener('click', function(event) {
	$( ".pics" ).empty();
	var submit = document.getElementById('sub').value;
	$.getJSON(url + submit + end, function(data) {
	console.log(data);
	$.each(data.data.children, function(i,object){
			$("<img>").attr("src", object.data.url).appendTo(".pics");
	});
	});
	});

The "document.getElementById("subsend").addEventListener('click', function(event)" is a method that says when you click the submit button it will run the code that is nested inside it. Because we want the site to "refresh" and be a blank slate everytime we append to images to it we use the .empty jQuery method which will remove all child nodes of the pics div. Next, we make a variable called submit which takes the text that was inputted by the user. Now, we put it all together with the variables url, submit, and end. We use the getJSON method on this combination, which loads JSON-encoded data from the server using a "GET" request. Now that we have the JSON loaded, we iterate through the data using the jQuery each function. The each function is a generic iterator that can iterate over both objects and arrays. The "data" is what we recieved from the getJSON request. The attr method that sets attributes to a selector. The selector in jQuery is used to find or select HTML elements based on their name, id, class, type, attribute, values of attribute and much more. In our case, we use it the HTML img tag as our selector and we give it the attributes of a src and url. After that we appendTo method to append images to the body. As a final result we can see all the images on teh front page of the subreddit in the results tab on jsfiddle.

Method #3 - Getting a Token and Account Information

In this section we will be using our secret and client ID to get a token. A token is maybe required to use any functions or methods that are account-based. Reddit uses OAuth2 authentication which is is a token based authentication standard and it allows third party services to access a user's accounts without exposing a user's password.

For this section we will be using a command line such as Terminal that allows you to use curl.

Click here to learn more about Outh2

Method #3 - Part 1

First open up your command line and enter the following line

curl -X POST -d 'grant_type=password&username=REDDIT_USERNAME&password=REDDIT_PASSWORD' --user 'CLIEND_ID:CLIENT_SECRET' https://www.reddit.com/api/v1/access_token

Replace the REDDIT_USERNAME with your username and REDDIT_PASSWORD with your account password. Then replace CLIENT_ID with your own client id and CLIENT_SECRET with your own secret. You can find both the CLIENT_ID and CLIENT_SECRET on the app tab of your preferences page. I will now show you an example.

The part highlighted in teal is where your password goes, the part highlighted in red is where your client ID goes and the part highlighted in blue is where your secret goes. Finally, the area that is circled in silver is your access token. howtophilip is my username.

Congratulations, now you should have your access token, let's get account information.

Please NOTE: The token only lasts for 1 hour. After 1 hour, you will have to request another token.

Method #3 - Part 2

Now that you have your token, let's get your account information. Input the command below to your command line to get your account information. Replace TOKEN with the token you recieved in part 1 and PROJECT with the name of your project and YourUserName with your reddit username

Method #4 - Overview of wrappers and snoowrap

What is a Wrapper?

A wrapper is essentially a program or a script that "wraps" the functions of another program. It generally, makes it easier to use the functions of the program that it wraps.


What is snoowrap?

snoowrap is a fully featured JavaScript wrapper for the reddit API. It provides a simple interface to access every reddit API endpoint.

Method #4 - Making the bot

Like before first create a new js file for your bot. Add the login code that was provided earlier and fill it with your own information

Below, I will show you the code to search for a specific query on /r/buildapcsales and message you the results

var all = new Array();
r.getSubreddit('buildapcsales').search({query: 'GPU', time: 'week'})
.then(function(sales){
  for (var i = 0; i < sales.length; i++){
    var stuff = new Array();
    stuff.push((sales[i].title) + "\n\n" + "[Link to Reddit Post]" + "(" + (sales[i].permalink)+ ")" + "\n\n" + "[Link to Sale]" +"(" +(sales[i].url)+ ")" + "\n\n")
    for (var j = 0; j < stuff.length; j++){
      all += stuff[j];}
    r.composeMessage({
      to: 'howtophilip',
      subject: "Your Daily Sales for GPUs",
      text: all  })}})

Method #4 - Explanation of Code

The first thing we do is create an array, in this case an array called "all", for the final body of text you want to send in the private message. In this case, we used a variable called all. Before we start explaining the next parts, it is important to know what a javascript promise is. When using the snoowrap wrapper, all the methods return a promise. A promise represents the eventual result of an asynchronous operation. In addition, a promise is convenient to write because you can do things asynchronously without the use of callbacks or what many refer to as "callback hell".

The getSubReddit method allows us navigate to the subreddit, we are using buildapcsales but you could replace this with any subreddit you would like. The search method then allows us to search through the method with the parameters of query and time. The query is what you want to find, we use GPU but again you can use something else, and time is the timespan of posts that you would like to retrieve from. Time could be hour, day, week, month, year or all.

Part 2 of Explanation
Learn More

Method #4 - Explanation of Code - Continued

The .then method is then used to return the promise. We then use the promise to iterate through the returned "sales", this is also a variable you could rename to whatever you want. We use a for loop to iterate through the "sales". In this for loop, we create a new array, "stuff" and push the title, reddit post link, and the link that the original reddit poster posted onto this array. The "\n\n" signifies new lines, while the brackets and parentheses are used because reddit uses a custom version of marksup. The brackets are used for a link name while the parenthesis are used for the link url.

We then use use another for loop and iterate through all the items we appended onto the array "stuff", and add everything onto the all array. Then nested inside the original for loop, we use the composeMessage method. This method has the parameters of of to, subject and text. The "to" is who you want to send the message too, in this case I sent it to myself, so it is my own username. The subject is what the messages intent is. And finally the text is the body of the message. In this case, it is all, it contains all the titles, links to the post, and the url the post links to.
